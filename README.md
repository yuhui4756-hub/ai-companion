# AI伴侣

一个面向个人和朋友使用的 BYOK（Bring Your Own Key，自带 API Key）AI 伴侣网页 Demo。

项目目标不是只做一个问答机器人，而是做一个具备长期记忆、情绪陪伴、人设切换、多模型接入能力的 AI 伴侣系统。第一阶段先完成网页 Demo，后续再预留 QQ、微信等社交入口接入。

## 核心方向

- 用户可选择不同类型的 AI 伴侣，如朋友陪伴、轻恋爱、理性支持、治愈陪伴、动漫/角色扮演等。
- 默认风格真实自然、有温度，不过度油腻，不像客服。
- 用户可自填主流 AI 服务商的 API Key，项目不内置、不代管商业 Key。
- 长期记忆是核心能力：记住称呼、偏好、重要事件、禁忌话题、关系进展和情绪模式。
- 用户必须能查看、编辑、删除长期记忆。
- 第一版只做中文网页 Demo，暂不做复杂注册登录。

## 第一阶段产物

- 产品定位和 MVP 范围。
- 多线程协作规则。
- AI 伴侣人设与记忆设计。
- BYOK 多模型接入蓝图。
- 可运行的网页聊天 Demo。

## 当前状态

项目初始化中。当前重点是先建立项目文档、线程职责和交接机制，然后进入网页 Demo 开发。
